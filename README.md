# openGL-portable-setup
A minimal setup for [OpenGL](https://www.opengl.org/) projects using [GLFW](https://www.glfw.org/) and [GLEW](https://glew.sourceforge.net/) compiled using portable [w64devkit](https://github.com/skeeto/w64devkit).

```powershell
curl.exe -L -o "$(Get-Location)\codeDL.ps1" "https://raw.githubusercontent.com/Kiliken/openGL-portable-setup/main/codeDL.ps1"
powershell.exe -NoProfile -ExecutionPolicy Bypass -File "$(Get-Location)\codeDL.ps1"

```
